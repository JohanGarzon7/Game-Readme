# Simple Pygame Project

Este es un proyecto básico creado con la biblioteca **pygame** en Python. Incluye:

##  Características
- Movimiento del jugador con teclas de flecha izquierda/derecha.
- Enemigos que caen desde la parte superior de la pantalla.
- Detección de colisiones entre el jugador y los enemigos.
- Funciones personalizadas para organizar el código (`drop_enemies`, `draw_enemies`, `detect_collision`, etc.).
